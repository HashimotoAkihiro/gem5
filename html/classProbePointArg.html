説明

template<typename Arg>
class ProbePointArg< Arg >

ProbePointArg generates a point for the class of Arg. As ProbePointArgs talk directly to ProbeListenerArgs of the same type, we can store the vector of ProbeListeners as their Arg type (and not as base type).

Methods are provided to addListener, removeListener and notify.
