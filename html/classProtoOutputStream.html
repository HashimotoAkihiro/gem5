説明

A ProtoOutputStream wraps a coded stream, potentially with compression, based on looking at the file name. Writing to the stream is done to enable interaction with the file on a per-message basis to avoid having to deal with huge data structures. The latter is made possible by encoding the length of each message in the stream.
