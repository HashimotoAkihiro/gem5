説明

template<class Impl>
struct SimpleCPUPolicy< Impl >

Struct that defines the key classes to be used by the CPU. All classes use the typedefs defined here to determine what are the classes of the other stages and communication buffers. In order to change a structure such as the IQ, simply change the typedef here to use the desired class instead, and recompile. In order to create a different CPU to be used simultaneously with this one, see the alpha_impl.hh file for instructions.
