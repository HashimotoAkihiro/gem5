int inCache

A bit mask of the sizes of cache that this block is resident in. Each bit represents a power of 2 in MB size cache. If bit 0 is set, this block is in a 1MB cache If bit 2 is set, this block is in a 4MB cache, etc. There is one bit for each cache smaller than the full size (default 16MB).
