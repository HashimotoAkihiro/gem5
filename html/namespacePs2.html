Table to convert simple key symbols (0x00XX) into ps2 bytes. Lower byte is the scan code to send and upper byte is if a modifier is required to generate it. The table generates us keyboard codes, (e.g. the guest is supposed to recognize the keyboard as en_US). A new table would be required for another locale.

Bitfield<2> middleButton
Bitfield<3> one
Bitfield<1> rightButton
