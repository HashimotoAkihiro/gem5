説明

Implementation specific struct that defines several key types to the CPU, the stages within the CPU, the time buffers, and the DynInst. The struct defines the ISA, the CPU policy, the specific DynInst, the specific O3CPU, and all of the structs from the time buffers to do communication. This is one of the key things that must be defined for each hardware specific CPU implementation.
